# Unminify & Minify

A simple and efficient tool for transforming code between **minified** and **readable** formats.
I created this tool because it became frustrating to rely on online minifiers and unminifiers that often fail to load or become slow - forcing me to manually minify or unminify code. This project solves that annoyance by providing a fast, reliable, offline solution.
